# 🎮 Game Store — Coursework: Backend + Test Automation Framework

## 📌 About the Project

**Game Store** is my **1nd year coursework** at BSUIR, where I developed a backend for an online game store. The main goal of the project is to build a **professional test automation framework** around it, demonstrating a systematic approach to API testing.

---

## 🧪 Test Automation Framework 

- **12 API tests** (JUnit 5)
- **REST Assured** + JSON Schema Validation
- **Java Faker** + Builder Pattern
- **AssertJ** + Custom Conditions
- **Parallel execution** (4 threads)
- **RetryListener** for flaky tests
- **Owner** for environment configuration
- **Allure reports** with steps and attachments
- **Docker Compose** (PostgreSQL)
